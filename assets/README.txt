assets usage notes

- Add your photo as file: /workspace/assets/profile.jpg
  Recommended square size, e.g. 600x600px. The app shows a fallback text
  until the file is present.

- Blog posts are stored locally in your browser using localStorage under key:
  "blogPosts". They are not uploaded to any server. Clearing browser data
  removes them.

- You can change the public contact e-mail by editing the link in index.html:
  element with id="contactEmailLink".
